## 📖 简介

妙算是一款轻量级多功能计算工具，基于 Flutter 构建。它将日常计算、单位换算与科学工具融为一体，使用毛玻璃界面设计，同时适配桌面与移动端。

## ✨ 功能

- **标准与科学计算器** 🧮 — 链式计算、历史记录、表达式编辑、复制粘贴
- **单位换算** 🔄 — 7 类换算（汇率、长度、面积、体积、温度、速度、质量）
- **实用工具** 🛠️ — 大写数字、进制转换、BMI 指数、亲戚称呼
- **方程求解** 📐 — 线性方程组求解
- **化学配平** ⚗️ — 自动配平化学方程式
- **函数绘图** 📈 — 绘制数学函数曲线
- **毛玻璃界面** 💧 — 毛玻璃设计，支持动态主题配色
- **响应式布局** 📱💻 — 自适应侧边栏、紧凑模式，360px 到 4K 皆可

## 🚀 安装

### Windows（MSIX）

1. 从 [Releases](https://github.com/Lordly-Tech/MiaoSuan/releases) 下载最新的 `.msix`
2. 将 `cert.cer` 安装到 **本地计算机\受信任的根证书颁发机构**
3. 运行 `install.bat`（自动提权）或双击 `.msix`
